# RL Poker Adaptive Opponent Profiler

This project explores how reinforcement learning can be used to adapt poker strategies by modeling opponent behavior.

The system analyzes opponent actions during simulated games and builds behavioral profiles based on patterns such as aggression, bluffing frequency, and folding tendencies. These profiles are then used to dynamically adjust the decision policy of a poker-playing agent.

The goal of the project is to investigate how adaptive strategies perform compared to static poker strategies when facing different types of opponents.

## Key Features

- Poker game simulation environment

- Opponent behavior profiling

- Reinforcement learning–based strategy adaptation

- Modular architecture for experimentation

- Configurable opponent strategies
